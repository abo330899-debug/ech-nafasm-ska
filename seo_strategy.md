# SEO Strategy

## In scope
- Public landing/login page at `/`
- Static HTML shell and public assets under `artifacts/nafsam/index.html` and `artifacts/nafsam/public/`
- Crawlability files for the deployed site (`robots.txt`, `sitemap.xml`, `llms.txt`)
- Social-sharing and AI-crawler visibility for the public route

## Out of scope
- Authenticated archive routes behind login (`/home`, `/photos`, `/journey`, `/songs`, `/videos`, `/writings`, `/feelings`, `/chat`)
- Admin-only reorder UI (`/api/reorder`)
- Private media and API payloads served only after authentication

## Target audience
- Known invitees or viewers of the Nafsam memory archive.
- Secondary audience for SEO purposes: anyone discovering the public archive landing page through search or shared links.

## Primary keywords
- Unknown.
- Likely branded intent around `Nafsam` and memory archive / personal archive queries.
